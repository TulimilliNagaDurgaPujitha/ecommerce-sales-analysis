# ecommerce-sales-analysis
E-commerce sales analysis using Python, SQL, Excel and Tableau. Analyzed 5000 orders across 6 categories and 5 regions in India.

## Overview
Analyzed 5,000 e-commerce orders to identify revenue trends, top categories, regional performance, and order status breakdown.

## Live Dashboard
[Click here to view](https://public.tableau.com/app/profile/naga.durga.pujitha.tulimilli/viz/E-commerceSalesAnalysis_17817624164420/E-commerceSalesAnalysisDashboard?publish=yes)

## Tools Used
- Python (pandas) — data cleaning and EDA
- SQL (SQLite) — business queries
- Google Sheets — pivot tables
- Tableau Public — interactive dashboard

## Key Findings
- Electronics is the top category with ₹8 crore revenue
- South region generates highest revenue
- Revenue grew from ₹2.1 crore (2022) to ₹4.2 crore (2024)
- 65% of orders are successfully delivered
- Jaipur is the top city by revenue

## Dataset
5,000 orders | 19 columns | 6 categories | 5 regions | 2022-2024
